About Kinalyzer:

Kinalyzer is a software suite developed to reconstruct sibling groups using genotypes from codominant markers such as microsatellites. Currently there are two algorithms available to reconstruct full-sibling groups for cases where parental genotypes are not available. Kinalyzer uses combinatorial optimization based on Mendelian inheritance rules to find the fewest number of sibling groups that contain all the individuals in the sample (‘2-allele set cover’). Also available is a ‘consensus’ method that reconstructs sibgroups using subsets of loci and finds the consensus of these different solutions.
